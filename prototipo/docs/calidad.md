# Aplicación de Calidad

Normas consideradas:

## ISO 9126

- Funcionalidad
- Usabilidad
- Fiabilidad
- Eficiencia
- Mantenibilidad
- Portabilidad

## ISO 25000

Casos de prueba aplicados:

- Inicio de sesión
- Registro de usuario
- Registro de vehículo
- Carga documental
- Validación de solicitudes
